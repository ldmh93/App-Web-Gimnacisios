# 5. Sistema de Ejercicios

> Parte de la arquitectura maestra de GymOS SaaS. Índice: [MASTER_ARCHITECTURE.md](./MASTER_ARCHITECTURE.md)

Debe existir una biblioteca central de ejercicios.

Cada ejercicio puede contener:

- Nombre.
- Descripción.
- Grupo muscular.
- Imagen.
- Video.
- Instrucciones.
- Errores comunes.
- Equipamiento necesario.

Las imágenes deben almacenarse usando **Supabase Storage**.
